# HomelyHub 🏡

A full-stack vacation rental platform inspired by Airbnb, built using the MERN ecosystem. Users can explore property listings, create and manage accommodations, upload images, leave reviews, and view property locations on interactive maps.

## 🚀 Features

* Secure user authentication and session management using Passport.js.
* Create, read, update, and delete property listings.
* Upload and manage listing images with Cloudinary.
* Review and rating system for properties.
* Interactive location mapping using Geoapify and OpenStreetMap.
* Server-side validation using Joi.
* Flash notifications for user feedback.
* MongoDB-backed session storage.
* Responsive UI built with Bootstrap and EJS templates.
